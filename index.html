Pasos para activar (una sola vez, gratis):
1. Abre console.firebase.google.com con tu cuenta Google
2. Crear proyecto → dale un nombre (ej: tiendas-2026) → Continuar
3. En el menú lateral: Realtime Database → Crear base de datos → Modo prueba → Listo
4. Ve a ⚙ Configuración del proyecto → Tus apps → Añadir app Web (</>)
5. Copia los valores apiKey, databaseURL y projectId y pégalos abajo
🔄 Google Sheets sync (importación)
⚠ IMPORTANTE — Publicar la hoja correcta:
1. En Google Sheets → Archivo → Compartir → Publicar en la web
2. En el primer desplegable selecciona RESUMEN GENERAL (¡no B-01, B-02, etc!)
3. En el segundo desplegable selecciona Valores separados por comas (.csv)
4. Clic en Publicar → copia la URL → pégala abajo
La URL correcta apunta a la hoja RESUMEN GENERAL y empieza con: https://docs.google.com/spreadsheets/d/e/2PACX-.../pub?output=csv
