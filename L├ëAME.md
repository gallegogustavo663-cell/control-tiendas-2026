# 🏪 Control de Tiendas 2026

Dashboard web en tiempo real para el control de novedades, faltantes, sobrantes y rendimiento de bodegas.

## 🚀 Acceso rápido

> **[Abrir Dashboard →](./index.html)**

## 📁 Estructura del repositorio

```
/
├── index.html          # Dashboard principal (PWA)
├── manifiesto.json     # Manifiesto PWA (instalable)
├── sw.js               # Service Worker (modo offline)
├── LÉAME.md            # Este archivo
└── iconos/             # Íconos de la app
    ├── icon-192.png
    └── icon-512.png
```

## ✨ Funcionalidades

- 📊 Resumen general con KPIs en tiempo real
- 💰 Panel financiero: faltantes, sobrantes y saldos
- 🏆 Ranking de rendimiento por responsable
- 🗂 Clasificación de novedades por bodega
- 📋 Gestión de pendientes
- 📝 Notas internas
- 📂 Importación de Excel (.xlsx / .csv)
- 📤 Exportación a CSV y PDF
- 🔄 Sincronización en tiempo real con Firebase
- 📱 Instalable como app (PWA)

## 📱 Instalar como app

En Chrome/Edge: abre el dashboard → menú `⋮` → **"Instalar aplicación"**

En iPhone/iPad: Safari → botón compartir → **"Agregar a pantalla de inicio"**

## 🛠 Tecnologías

- HTML5 / CSS3 / JavaScript vanilla
- [Chart.js 4.4](https://www.chartjs.org/)
- [Firebase Realtime Database](https://firebase.google.com/)
- [SheetJS](https://sheetjs.com/) (lectura de Excel)
- Service Worker (PWA offline)
